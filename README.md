# SWE 250P LAB A: Web Programming


## Course Description

This course provides a technical overview of the fundamentals of modern web application development.
It covers key concepts, terminology, and popular frameworks used to build full-stack web applications.

Throughout the course, students work with technologies such as **HTML/CSS, Node.js, React, and TypeScript**, and learn how to build, bundle, and deploy modern web applications.
The course also includes the development of a complete email web application.

---

## Repository Description

This repository contains **my course assignments for SWE 250P LAB A**.

The project is organized into **frontend** and **backend** components, following a typical full-stack web application structure:

* **Frontend:**
  Responsible for the user interface and client-side logic.

* **Backend:**
  Responsible for server-side logic, APIs, and data handling.

Each part corresponds to the lab assignments and exercises required for this course.
